对于那些不太熟悉 R 编程语言的人来说，这里有一个概述。

r 是一种函数式语言，它为统计计算和图形提供了一个自由的软件环境。r 提供精心设计的出版物质量的绘图，包括数学符号和公式。

### **R 的特性**

1.  数据处理和操作。
2.  用于数组计算的运算符。
3.  一大堆相关的工具。
4.  数据分析的图形设备。
