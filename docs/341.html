什么是Swift？

Swift是一种通用编程语言。苹果为“I”设备开发了它——iPod、iOS、macOS等。以及Linux。它是一种编译语言，旨在与苹果的Cocoa和Cocoa Touch框架一起工作。Swift使用Objective-C运行时库，该库允许Swift、C、C++和Objective C代码在单个程序中运行。来源- 维基百科

雨燕难学吗？

嗯，苹果确实声称Swift很容易学，而且与许多其他语言相比也是如此。只有在你没有任何编程经验的情况下，Swift才和任何编程语言一样难。如果你能掌握编程语言的基本概念，Swift应该相当容易学习——它庞大而复杂，但并非不可能学习。和其他语言一样，你需要练习和经验。你可以考虑学习Swift与C++或Scala等语言平起平坐。

为什么要学Swift？

学习Swift有很多原因。首先，它对于移动应用程序的开发来说既简单又有用。它是一种通用的语言。

Swift是作为“每个人都可以编码”计划的一部分开发的，这意味着该语言足够简单，每个人都可以开始编码。

我们已经知道，iPhones、iPad等。是世界上最畅销的设备，还有什么比开发iPhone应用程序更好的呢？如果您了解Swift，您可以使用内置的直观工具快速开发应用程序。

Swift是开源的，拥有出色的社区支持。由于Swift相对较新，因此有很大的创新和合作空间，甚至您也可以做出贡献！

最重要的是，对Swift开发人员的需求巨大，一名Swift开发人员的平均年薪可高达85，000美元！

学雨燕需要多长时间？

学习节奏因人而异。

如果你完全没有编程经验，花点时间熟悉编码术语和学习任何编程语言的基本概念。比如OOP、构建、部署、编译等。虽然你可以通过一些好的教程和书籍来加快学习速度，但如果你打算自学，那会增加你的时间。作为一个普通的学习者，如果你有一些编程经验，你将能够在大约3-4周内编写简单的Swift代码。

如果你是大一新生，我建议你从网上一些好的教程开始——包括免费和付费的。你需要一台笔记本电脑和必要的软件来练习。

没有苹果电脑有可能学Swift吗？

虽然没有苹果电脑也有学习Swift的变通办法——比如在OSX或 Hackintosh 上使用虚拟机，但你可以买一台便宜的转售并用来练习，因为学习Swift的主要目的是在“I”平台上开发应用程序！你可以买一台翻新的Mac mini——这将节省你大量的时间，你可以专注于学习Swift，而不是做黑客修理你的电脑。但是，如果你想走更长的路——去虚拟盒子或Hackintosh。

如何学习Swift？

学任何东西都没有捷径。如果你是有几年经验的程序员，可以跟着Youtube上的基础教程和视频。您也可以查看Swift的官方网站，了解文档和支持。无论你走哪条路，关键是跟着线上和线下的材料一起练习。

如果你以前没有任何编程经验，你应该从官方文档和他们提供的示例以及网上的基础教程开始。除此之外，试试这本好看又容易理解的基础书籍开始你的学习。

如果您熟悉Java或JavaScript等语言，您应该能够通过以下简单代码掌握Swift的窍门:

关于这个简单片段的一些事情:

/* This program will print your name */
var myName = “TypeYourName”
print(myName)

每行之后不需要分号；但是，如果你真的把它放了，编译器不会生你的气。

  • Swift通过/**/支持多行注释，通过//支持单行注释
  • 注意命名约定；这个单词以一个小写字母开头；然而，同一个变量名中的第二个单词以大写字母开始- m y N ame
  • 你不需要声明变量的类型var就可以了！

我们在上面了解到，只给出‘var’就可以了；然而，即使您确实给出了类型，编译器也是非常宽容的:

常量是用let声明的

var a: Int = 10

执行类型转换很容易:

let cons = 3.14

就像任何其他编程语言一样，Swift支持算术、逻辑、赋值、位运算等运算符。举个例子，

var a: Int = 20
var b: Float = 10.33
print(“Float to Int: Int(b)”)
print(“Int to Float: Float(a)”)

将输出为20。

print(“(10+10)”)

Swift中使用条件语句来决定是否满足某个条件。

一个简单的for循环如下所示:

if varx < 5{
print(“varx is less than five”)
} else if varx > 20 && varx < 45 {
print(“varx is between 21 and 44”)
}else {
print(“varx is greater than 44”)
}

While和do-while也存在于Swift中，并且具有类似于Java的语法。

for j in 1…5{
print(“the value of j is (j)”)
}

除此之外，Swift还支持数组和元组、集合、字典、结构、枚举、闭包和函数。Swift也支持OOP，你可以添加自己的类和方法。您可以将类别定义为:
